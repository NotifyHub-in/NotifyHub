## Who These Guides Are For

- platform operators
- service owners integrating an upstream application
- developers testing email, SMS, WhatsApp, push, or webhook delivery

## What These Guides Assume

- the control plane is running locally or in an environment you can reach
- you can call the API
- you have the provider credentials or secret references for the channels you want to enable
