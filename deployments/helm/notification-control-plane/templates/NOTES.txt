Release flow:

4. Run a smoke notification request after the rollout.

Rollback guidance:
- Use `helm rollback` for app/config regressions.
- For schema changes, prefer a forward fix if the latest migration is not rollback-safe.
- Re-run the migration job with the corrected release once the code or schema is fixed.
